# Image Assets

Please add your personal photos to this folder to use them on the website.
For best performance and layout, consider the following image names or update the code in `ModellingGallery.jsx` to match your actual file names.

### Suggested File Names
1. `portrait-1.jpg` (Main hero/about portrait)
2. `fashion-1.jpg` (Modelling section)
3. `fashion-2.jpg` (Modelling section)
4. `fashion-3.jpg` (Modelling section)
5. `project-1.jpg` (PitchPerfect cover)
6. `project-2.jpg` (Fruit Ninja cover)

**How to use them in code:**
```javascript
import portrait1 from '../assets/images/portrait-1.jpg';

// Inside JSX:
<img src={portrait1} alt="Shone Mariyam Shery" />
```

Currently, the code uses premium placeholders from Unsplash or simple colored blocks to demonstrate the layout.
